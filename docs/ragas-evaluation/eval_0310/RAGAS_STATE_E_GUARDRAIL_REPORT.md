## 6. 결론

### 가드레일 변경은 실패 — 코드 롤백 필요

키워드 가드레일 + LLM 분류 프롬프트 강화 + BM25 검증 변경이 **모든 RAGAS 메트릭을 악화**시켰습니다.

**근본 원인 추정:**
1. LLM 도메인 분류 변경으로 잘못된 컬렉션에서 검색 → 관련 없는 컨텍스트 반환
2. Generator가 관련 없는 컨텍스트를 받아 "확인할 수 없습니다" 보수적 답변
3. 프롬프트 변경이 LLM 분류 정확도를 낮춘 가능성

### 후속 조치

1. **즉시**: A+B+C 변경 롤백 (commit `94a407f` revert)
2. **분석**: 실패 8건의 실제 분류 결과 로깅 확인 (어느 도메인으로 라우팅되었는지)
3. **재설계**: 키워드 가드레일을 LLM 분류 *보완*이 아닌 *검증*으로 사용하는 방식 재고
4. **테스트**: 개별 질문 단위 CLI 테스트로 분류 동작 확인 후 배치 평가 재실행
