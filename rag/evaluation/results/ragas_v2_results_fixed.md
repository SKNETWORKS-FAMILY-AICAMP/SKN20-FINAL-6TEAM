## 주요 발견사항

### 1. Faithfulness / Answer Relevancy가 낮은 원인
- **LLM 자체 지식 의존**: GPT-4가 검색된 컨텍스트보다 자신의 학습 데이터를 기반으로 답변하는 경향
- **한국어 NLI 한계**: RAGAS의 NLI(자연어추론) 평가에서 동일한 의미를 다르게 표현하면 불일치로 판정
- **답변 길이**: 긴 답변에서 많은 진술문이 생성되고, 대부분 컨텍스트에 직접 매칭되지 않아 점수 하락

### 2. Context Precision이 높은 이유
- Hybrid Search(BM25 + Vector + RRF) + Cross-Encoder Reranking이 효과적으로 작동
- 관련 문서를 상위에 잘 배치함

### 3. 도메인별 편차
- **재무/세무** 및 **세무+법률** 복합 질문에서 상대적으로 높은 Faithfulness
- **인사/노무**, **법률** 단독 도메인에서 Faithfulness 0.0 - 법률 텍스트의 표현과 LLM 답변 표현 차이가 큼
- **종합 복합 질문**(3개 이상 도메인)에서 전반적으로 낮은 점수

### 4. 개선 방향
- Faithfulness 향상: 프롬프트에서 "검색된 문서를 근거로 답변하라"는 지시 강화
- NLI 평가 개선: 한국어 의미 동치성을 더 잘 판단하도록 NLI 프롬프트 추가 조정
- 복합 도메인 검색 개선: Cross-Domain 검색 시 각 도메인별 문서 배분 최적화
