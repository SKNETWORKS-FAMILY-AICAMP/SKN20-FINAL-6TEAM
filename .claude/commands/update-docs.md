기능 구현 후 CLAUDE.md와 AGENTS.md를 자동으로 갱신합니다.

## 절차

1. `git diff --name-only HEAD~1` 또는 `git diff --cached --name-only`로 변경된 파일 목록 확인
2. 변경된 파일이 속한 서비스 식별 (backend/, frontend/, rag/, data/)
3. 해당 서비스의 CLAUDE.md 읽기
4. 변경 사항 분석:
   - 새로운 파일이 프로젝트 구조에 반영되었는지 확인
   - 새로운 API 엔드포인트가 문서에 있는지 확인
   - 새로운 모듈/컴포넌트가 설명에 포함되었는지 확인
   - 환경 변수 변경이 반영되었는지 확인
5. 누락된 항목을 CLAUDE.md에 추가
6. 해당 서비스의 AGENTS.md도 동기화 (요약본 업데이트)
7. 루트 CLAUDE.md의 프로젝트 구조도 필요시 업데이트

## 출력
변경된 문서 목록과 추가된 항목을 요약하여 보고합니다.
변경이 불필요한 경우 "문서가 최신 상태입니다"라고 알립니다.
