# Claude Code Skills — Bizi Project

프로젝트에서 사용하는 on-demand 스킬. Claude Code가 관련 작업 시 자동으로 로드합니다.

## 스킬 목록

| 스킬 | 트리거 | 설명 |
|------|--------|------|
| `code-patterns/` | 백엔드/프론트엔드 코드 작성 | SQLAlchemy 2.0, FastAPI, Zustand, LangChain 패턴 |
| `test-guide/` | 테스트 구조 설계, 커버리지 개선 | pytest/Vitest 구조, 커버리지 목표(75%+), RAGAS 평가 |
| `fastapi-endpoint/` | API 엔드포인트 생성 | 라우터, 서비스, 스키마 보일러플레이트 |
| `pytest-suite/` | pytest 테스트 스위트 생성 | TDD 패턴 테스트 코드 생성 |
| `react-component/` | React 컴포넌트 생성 | TypeScript + TailwindCSS + Vitest |
| `rag-agent/` | RAG 에이전트 생성 | BaseAgent 상속, 벡터 컬렉션 연결 |
| `implement/` | 계획 파일 기반 구현 | 계획 모드 없이 즉시 코드 적용 |
